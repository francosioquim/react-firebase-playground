# Crab Cake Entree

### Getting Started
Follow this [guide](https://app.tettra.co/teams/roamltd/pages/how-to-create-a-new-fe-project) on Tettra

### Usage

Start the App

```bash
yarn start
```

Create build files

```bash
yarn build
```

Deploy to Github pages

```bash
yarn deploy
```
